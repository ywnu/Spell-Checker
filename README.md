# Smart Spell Checker

A Python-based Smart Spell Checker application built for Data Structure Course - CSC 3302.

## How It Works
1. Load the dictionary file
2. User enters text in the GUI
3. System searches each word in the Trie
4. Generates suggestions for incorrect words
5. Displays corrected sentence

## How to Use

### Requirements
pip install pyspellchecker

### Run the Project
python main.py

### Steps
1. Open the application
2. Wait for the dictionary to load (370,105 words)
3. Type your text in the Enter Text box
4. Click Check Spelling
5. View results and corrected sentence
6. Click Clear to start over

## Algorithms & Data Structures
- Trie (store & search words)
- Spellchecker (generate suggestions)
- Regular Expressions (extract words)
- Threading (keep GUI responsive)

## Technologies
- Python
- Tkinter (GUI)
- spellchecker library
